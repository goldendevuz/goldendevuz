## About

Hi, I'm **Abdulmajid Yunus**, a software engineer, open-source enthusiast, and founder of **Univel**.

I enjoy turning ambitious ideas into real products.

Today my primary focus is building AI-powered software, creating developer tools, publishing open-source projects, and launching startups that solve meaningful problems.

Instead of building one company around one product, I'm building an ecosystem where products, developers, founders, and communities can grow together.

---

## About Univel

**Univel** is an open-source ecosystem focused on building AI products, developer tools, and useful software.

Every product begins with a real problem.

Every solution starts as open source.

Every successful project can evolve into a sustainable business.

The long-term vision is simple:

> Build the home of open-source AI startups.

---

## Mission

> **Building AI products, open-source software, and startups that solve real problems.**

I believe software should create more value than it captures.

Knowledge should be shared.

Developers should be able to learn from real production projects instead of isolated examples.

Open source isn't a marketing strategy.

It's the foundation.

---

## Vision

I want **Univel** to become the home for open-source AI startups.

A place where founders can launch products faster.

Where developers learn from real software.

Where communities improve products together.

And where open source becomes the starting point—not the finish line.

---

## Why Open Source?

Most software teaches people how to use products.

Open source teaches people how to build them.

Every project I publish is an investment in future developers, founders, and creators.

If someone can learn from my code, improve it, or even build a successful business on top of it, then the project has achieved something bigger than software.
